#  Artraid ML: Предиктивная модель выкупа заказов

Решение задачи прогнозирования вероятности выкупа заказа для компании **Artraid** (розничная продажа медицинских изделий).

**Цель проекта:** Разработать ML-модель, которая в день оформления заказа оценивает вероятность выкупа. Это позволяет бизнесу заранее сегментировать заказы и применять стратегии удержания для рискованных клиентов.

### 🔑 Ключевые возможности решения
*   **Прогнозирование:** Классификация заказов (Выкуп / Отказ) с использованием градиентного бустинга (CatBoost).
*   **Защита от Data Leakage:** Тщательная обработка временных меток и исключение признаков, недоступных в момент заказа.
*   **Feature Engineering:** Парсинг текстовых данных о составе заказа и извлечение временных признаков.
*   **Интерпретируемость:** Анализ важности признаков с помощью SHAP.
*   **Финансовая симуляция:** Расчет ожидаемой прибыли и сравнение с базовой стратегией (отправка 100% заказов).

## 🧠 Логика модели
### Обработка утечек данных (Data Leakage)
Критически важным шагом было исключение признаков, которые становятся известны только после отправки заказа (например, handed_to_delivery_ts, rejected_ts, статусы оплаты). Использование этих данных дало бы завышенные метрики, но сделало бы модель бесполезной в проде.
### Выбор алгоритма
Использован CatBoost из-за его способности нативно работать с категориальными признаками (город, служба доставки, менеджер) без One-Hot Encoding, что сохраняет информативность данных и ускоряет обучение.
